## Usage

### Setup and Basic Usage

First, clone the repo and install the dependencies:

```shell
git clone https://github.com/shibil-rahman/easy-vqa-revolvers.git
cd easy-vqa-revolvers/
```

To run the model,

```shell
python train.py
```
